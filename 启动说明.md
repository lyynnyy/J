# 启动说明

## 问题排查

如果遇到"访问被拒绝"的问题，请按以下步骤操作：

### 1. 清理端口

```bash
# 查找占用5000端口的进程
lsof -ti:5000

# 如果找到进程，杀死它们
kill -9 $(lsof -ti:5000)
```

### 2. 启动应用

```bash
cd /Users/lynn/智能体/Agent_v4_多工具/poc_demo
python3 web_app.py
```

### 3. 访问应用

在浏览器中访问：`http://127.0.0.1:5000` 或 `http://localhost:5000`

### 4. 如果仍然被拒绝

- 确保数据库已创建：运行 `python3 db_build.py`
- 检查防火墙设置
- 尝试使用其他端口（修改 web_app.py 中的 port=5001）

## 常见问题

1. **端口被占用**：使用 `lsof -ti:5000` 查找并杀死占用进程
2. **数据库不存在**：运行 `python3 db_build.py` 创建数据库
3. **权限问题**：确保有读取文件的权限
